## üî¢ 2. GCD Using while (Euclidean Algorithm)

### In-Class Practice
Write a program that reads **two numbers** and computes their **GCD** using a `while` loop.

The **Euclidean Algorithm** was applied step-by-step:
- repeatedly replace the larger number with the remainder,
- continue until the remainder becomes zero.

This reinforced:
- loop-based problem solving,
- using conditions to control repetition.

## üîÇ 3. for Loop ‚Äî Concept and Structure

It was explained that in many problems, we already know **how many times a loop should run**.  
In such cases, using a `for` loop is often **cleaner and more readable** than `while`.

The structure of the `for` loop was explained conceptually:

- **Initialization (I)** ‚Äî executed once at the beginning  
- **Condition (C)** ‚Äî checked before each iteration  
- **Update (U)** ‚Äî executed after each iteration
